# 快速开始

## 应用功能区介绍

![Add new](/img/anyui-layout.svg)

## 启动应用

![Add new](/img/anyui-home.svg)

## 创建项目

> 点击"+"新建项目，然后根据向导选择：开发板类型、应用模板、项目名称、颜色位深、屏幕分辨率

![Add new](/img/anyui-create-new.svg)

## 设计界面

> 拖拽组件到画布区

## 保存项目

> 点击"保存"按钮

## 生成代码

> 点击"生成代码"，会提示是否覆盖之前生成的代码，“确认”后便会生成最新代码。

## 编译运行

:::warning
Web版本暂不支持编译运行，请下载桌面版。
:::
