模板组件

模板组件是用户组件模板。用户可以将修改过属性和样式的组件保存为模板组件。

模板组件可以包含子组件，例如：在一个容器组件中包含其它组件，可以将整个容器组件保存为模板组件。

保存模板组件​

选择组件 -> 右键 -> 保存为模板组件。

用户可以像使用普通组件一样使用模板组件。

修改模板组件​

对已保存的模板组件，可以在模板组件设置里修改名称和删除模板组件。

warning

模板组件只包含组件属性和样式，但不包含事件。
